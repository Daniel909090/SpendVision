# SpendVision
SpendVision is an Android application that transforms receipt images into structured financial intelligence. Products are automatically grouped into categories such as groceries, transport, utilities, clothing, etc. Food purchases are further analysed into subcategories, including fruit, vegetables, snacks, drinks, and unhealthy foods. Users can also track recurring financial commitments such as mortgages, subscriptions, loans, and car leasing agreements, including repayment timelines and remaining balances.
Users can scan or upload receipts directly through the mobile app.
The system uses OCR technology to extract store names, dates, products, quantities, and prices from receipts. 
All data is stored in a database and displayed through dashboards showing spending trends, category breakdowns, long-term financial commitments, and historical item price changes over time.
